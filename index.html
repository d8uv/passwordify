Passwordify!

Most people, by their own nature, generate shitty passwords. Passwordify is a tool that makes really strong passwords for you. Just type in a short master password, the name of the site you want to make a password for, and you'll get an incredibly strong password that is almost entirely unbreakable.

  • Note, this doesn't need to be very long. Something like "ff" will provide the same amount of security as "7Okm2MA1YgUnakUmbTYRYQ==".
